(static) middleware()

A middleware generator function. It allows access if at least one of the provided authenticators returns true or a promise that resolves to true. Authenticators take an "authdata" object wich contains the token from the header and the username, in case the header has the form user:hash. They also take an "identify" functions wich takes an identification of the client. Wathever object this function takes is the "me" object in calls to the next apify-wrapped middleware.
