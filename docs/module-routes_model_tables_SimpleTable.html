(inner) SimpleTable(name, fields, primaryKeys)

A PostgresTable with a much simpler interface.
Parameters:
Name Type Description
name String
fields Object
primaryKeys Array INTERFACE: create: adds a new record/row read: gets a list of all rows whose properties match those of the json object. If no filter is specified, all rows are returned. exists: returns true or false depending on wether the corresponding read would return any row. update: searches rows according to the first JSON safe object and modifies the fields of the second argument delete: deletes all rows matching the JSON object. If no parameter is passed, deletes all rows.
